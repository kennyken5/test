# chap.00 プロローグ

冒頭のINVALIDジョークについて、2014年に追記。
"INVALID SYSTEM DISK"というメッセージは、
起動時にシステムディスクではないものがFDDに検出されたときに表示されるもの。
2005年ではＦＤＤにデータディスクいれっぱで電源入れてピー！とエラーするのがありえた。
2014年でこれが起こるすれば、起動HDD破損とかになるので笑いごとではない。
移り変わりが激しいのである。
